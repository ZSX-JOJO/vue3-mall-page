# vue3-mall
基于脚手架vue-cli3.0的商城静态页面，使用vue3的语法进行编写。
特此声明：本源码仅供学习交流使用，不得用于违法用途，如果侵犯了国家法律，责任自负。

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
